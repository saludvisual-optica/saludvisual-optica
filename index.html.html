Configuración inicial

Sigue estos pasos para conectar el formulario con tu Excel en OneDrive

1

Registra una aplicación gratuita en Microsoft Azure

Ve a portal.azure.com → App registrations e inicia sesión con tu cuenta Microsoft.
Haz clic en + New registration y llena:

• Name: Servicios de Salud
• Supported account types: Personal Microsoft accounts only
• Redirect URI: Single-page application (SPA) →

Calculando...

Haz clic en Register.

2

Copia tu Application (client) ID

Después de registrar, verás el Application (client) ID en la pantalla de Overview. Cópialo y pégalo aquí:

3

Sube el formulario a GitHub Pages (una sola vez)

1. Crea cuenta gratis en github.com
2. Crea un repositorio nuevo → nómbralo ordenes
3. Sube este archivo index.html al repositorio
4. Ve a Settings → Pages → Branch: main → Save
5. En 1 minuto tendrás tu URL permanente tipo:

https://tuusuario.github.io/ordenes/

⚠️ Actualiza el Redirect URI del paso 1 con esa URL.

4

Inicia sesión y elige tu archivo Excel

Una vez guardado el Client ID, inicia sesión con tu cuenta Microsoft y selecciona el archivo CLIENTES_MEJORADO.xlsx de tu OneDrive.
